# CRONOS (Compact Rocket Outlining and Numerical Optimization Simulator)
Simplified rocket design and flight simulator intended for amateur liquid rocketry. Will include a 3-DoF trajectory module incorporating supersonic drag effects, low-fidelity internal component organization module, and low-fidelity propulsion module with hooks to interface with more advanced propulsion models.

# Name
In ancient Greek mythology, Cronos (also spelled Cronus or Kronos) was the titan of time, the eldest son of the earth Gaia and sky Uranus.
